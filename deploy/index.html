                # 概述

                此项目是基于Adachi-BOT的频道移植版

                原项目地址：Adachi-BOT (opens new window)

                # 频道机器人

                频道机器人由腾讯官方推出用于新版群聊（感觉更像小社区）

                与QQ群机器人相比，不会无缘无故被冻结，

                但发送消息也有颇多限制，用于本项目还是绰绰有余

                # 原项目文档

                Adachi-Docs (opens new window)

                这是原项目的官方文档地址，以下是频道BOT使用不同的部分

                # 安装及启动

                # Windows

                需要提前装好nodejs 14+的版本 和 安装开启redis服务，端口56379

                git clone https://ghproxy.com/https://github.com/Extrwave/Adachi-GBOT.git
cd Adachi-GBOT
npm install
npm run start

                初次启动会生成config文件夹，需要配置好setting.yaml和cookie.yaml才可以使用

                # 更新

                cd Adachi-GBOT
git pull

                请注意如果本地有更改需要先提交再次更新

                cd Adachi-GBOT
git add .
git commit -m "my modify"
git pull

                遇到各种冲突建议放弃本地更改更新，或者自己有能力解决冲突

                cd Adachi-GBOT
git reset --hard head
git pull

                # Docker

                git clone https://ghproxy.com/https://github.com/Extrwave/Adachi-GBOT.git
cd Adachi-BOT

docker-compose up -d
docker-compose restart

                # 更新

                # 注意需在 Adachi-BOT 根目录中运行命令
git pull
docker-compose restart
# 如果涉及更新Dockerfile文件则需要docker-compose down 然后 docker-compose up -d --build

                # PM2

                git clone hhttps://ghproxy.com/https://github.com/Extrwave/Adachi-GBOT.gitttps://ghproxy.com/https://github.com/SilveryStar/Adachi-BOT.git

cd Adachi-BOT

npm install -g pm2

# 初次运行将生成配置文件模板
# 根据右上角「配置」选项中的信息配置 setting.yml
npm run pm2-start

# 确保你已经在 6379 端口上运行 Redis 数据库
