      # 概述

      此项目是基于Adachi-BOT的频道移植版

      原项目地址：Adachi-BOT (opens new window)

      # 频道机器人

      频道机器人由腾讯官方推出用于新版群聊（感觉更像小社区）

      与QQ群机器人相比，不会无缘无故被冻结，

      但发送消息也有颇多限制，用于本项目还是绰绰有余

      # 原项目文档

      Adachi-Docs (opens new window)

      这是原项目的官方文档地址，以下是频道BOT使用不同的部分

      # 安装及启动

      暂时只测试了Windows环境，其余自测，有问题可以github提issue，或者下方评论留言

      需要提前装好nodejs 14+的版本 和 安装开启redis服务

      git clone -b ossdev https://github.com/Extrwave/Adachi-GBOT.git Adachi-GBOT
cd Adachi-GBOT
npm install
npm run start

      初次启动会生成config文件夹，需要配置好setting.yaml和cookie.yaml才可以使用

      # 更新

      cd Adachi-GBOT
git pull origin ossdev

      请注意本地更改需要先提交再次更新

      cd Adachi-GBOT
git add .
git commit -m "my modify"
git pull origin ossdev

      遇到合并冲突建议放弃本地更改更新，或者自己有能力解决冲突

      cd Adachi-GBOT
git reset --hard head
git pull origin ossdev
